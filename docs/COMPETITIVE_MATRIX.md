# Competitive matrix — Aegis vs IDV & adjacent players

> Provisional scores (1–5) for strategy, not marketing claims. Horizontal IDV vendors win on coverage; Aegis wins on vertical depth.

## Summary thesis

**Do not out-Trulioo Trulioo.** Use 1–2 IDV vendors as the identity base layer. Own role-specific risk, UK compliance evidence packs, Trust & Safety ops, and service orchestration for high-stakes human services. Create reverse dependency: platforms and vendors embed Aegis for care/concierge/security workflows.

## Gap Aegis fills

Generalist IDV optimises for fintech, crypto, marketplaces, and e-commerce onboarding. High-stakes **household access**, **executive discretion**, and **care/safeguarding** need:

1. Category compliance matrices (not one-size KYC)  
2. Manual clearance with audited rationale  
3. Continuity of engagement (bookings, logs, incidents)  
4. Liability-aware product copy and ops playbooks  

## Partnership model

```
[ Trulioo / Socure / … ]  →  base IDV result
           ↓
[ Aegis wallet + matrix + AI triage + human CLEAR ]
           ↓
[ Match · NDA · booking · service evidence · KPIs ]
```

Inbound over time: IDV vendors or staffing platforms request Aegis orchestration as the vertical layer.

## What would change scores upward for Aegis

- Live production IDV integration (coverage via partner, not rebuild)  
- Real DBS/SIA verification workflows (registered body / licence APIs where lawful)  
- Proven incident rate &lt; 0.5–1 / 1k bookings in pilot  
- White-label agency deployments
